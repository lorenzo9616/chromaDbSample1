# Technology Stack

- **.NET 8 Blazor Web App** — Interactive server-side rendered web UI
- **C#** — Primary programming language
- **ChromaDB.Client** — .NET client library for interacting with ChromaDB
- **Docker** — Containerization of the Blazor application
- **Docker Compose** — Orchestration of ChromaDB and the Blazor web app
- **HTML/CSS** — Frontend markup and styling
